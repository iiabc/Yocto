# Yocto 世界保护插件

## 简介

Yocto 是一款旨在提供**开箱即用**世界保护机制的 Minecraft 插件。它通过一系列全面的保护措施，有效防止玩家或生物对服务器世界进行未经授权的修改和破坏，同时
**允许拥有特定权限的玩家绕过保护**。

-----

## 构建与配置

### 构建发行版本

```
./gradlew build
```

## 权限设置

拥有 `yocto.bypass` 权限的玩家可以**绕过**所有保护机制。

-----

## 核心功能

Yocto 插件提供了以下全面的世界保护功能：

* **生物行为限制**
    * 禁止生物繁殖
    * 禁止生物修改方块
    * 禁止生物踩坏农田
    * 禁止实体爆炸
    * 禁止特定传送方式（如末影珍珠）
* **方块与环境保护**
    * 禁止方块爆炸（如TNT）
    * 禁止方块蔓延（如菌丝、草方块）
    * 禁止树叶消退
    * 禁止挂饰（如画、告示牌）被破坏
* **玩家行为限制**
    * 禁止鱼竿移动盔甲架
    * 禁止创造模式下射出的弓箭在世界中停留
    * 禁止玩家交互特定方块
    * 禁止打开特定页面
    * 禁止合成物品
    * 禁止配方补全
* **特殊保护**
    * 虚空保护
    * 防止生物入侵
    * 禁止植物生长（如树木）
    * 屏蔽成就消息
    * 禁止实体受到伤害
